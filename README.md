# **React Assignment Day 4 Quiz**

## **1. What is the primary purpose of the useEffect hook?**  
**Answer:** b) To handle side effects in functional components

---

## **2. When does useEffect with an empty dependency array run?**  
**Answer:** b) Only once after the first render (mount)

---

## **3. What is the correct way to fetch data with useEffect?**  
**Answer:** d) Both b and c are correct

---

## **4. Why do we need cleanup functions in useEffect?**  
**Answer:** b) To prevent memory leaks and cancel ongoing operations

---

## **5. What happens if you forget to include a dependency in the dependency array?**  
**Answer:** b) The effect will use stale values

---

## **6. How do you update the document title in a useEffect?**  
**Answer:** a) `document.title = "New Title"`

---

## **7. Which pattern creates an infinite loop?**  
**Answer:** c) `useEffect(() => { setState(value + 1) })`

---

## **8. How do you clean up a timer in useEffect?**  
**Answer:** c) `return () => clearInterval(timerID)` from the effect

---

## **9. When should you use multiple useEffect hooks?**  
**Answer:** b) To separate different concerns

---

## **10. What does the dependency array control?**  
**Answer:** a) When the effect runs
